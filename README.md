## Descripción General

Este repositorio contiene todas las prácticas, ejercicios y desafíos desarrollados a lo largo de la materia. El objetivo principal es aplicar los fundamentos de programación en Python, desde estructuras de control básicas hasta manipulación de datos y estructuras complejas (como listas, diccionarios, tuplas y dataframes usando Pandas).

---

## Estructura del Proyecto

El repositorio está organizado en distintas carpetas, cada una correspondiente a una clase o a una actividad específica evaluada durante el curso:

```text
ECBD-Python/
├── Actividad-II/   # Ejercicios prácticos con estructuras de control
├── Actividad V/    # Desafíos con estructuras de datos complejas y Pandas
```

---

## Contenido por Carpetas

A continuación se detalla de manera general lo que se aborda en cada una de las carpetas del repositorio.

### Actividad-II

Esta carpeta contiene ejercicios enfocados en la lógica de programación y estructuras de control básicas.

- **Conceptos aplicados:** Bucles (`for`, `while`), condicionales (`if / elif / else`), listas y diccionarios.
- **Proyectos clave:** Sistema de control de calificaciones, gestión interactiva de inventario y análisis estadístico lógico sin funciones built-in.

### Actividad V

Contiene ejercicios más avanzados orientados a la gestión de datos estructurados y manipulación de información.

- **Conceptos aplicados:** Listas, tuplas, sets, diccionarios avanzados y análisis de datos utilizando **Pandas** (`DataFrames`).
- **Proyectos clave:** Desafío integrador de un "Sistema de Gestión de Videojuegos", donde se procesan ventas, ingresos y estadísticas generales de un catálogo mediante Pandas.

### clase-1

Ejercicios introductorios desarrollados durante la primera sesión de la materia.

- **Conceptos aplicados:** Sintaxis básica de Python, declaración de variables, tipos de datos, entrada por consola (`input()`) y operaciones matemáticas elementales.

### clase-2

Prácticas de la segunda sesión, aumentando gradualmente la complejidad.

- **Conceptos aplicados:** Formateo de cadenas de texto (`f-strings`), operadores lógicos, estructuras condicionales simples y primeros acercamientos a colecciones de datos.

## Requisitos y Ejecución

Para ejecutar cualquiera de los scripts de este repositorio, asegúrate de cumplir con los siguientes requisitos:

- **Python 3.x** instalado en tu sistema.
- **Librería Pandas** (solo requerida para la `Actividad V`). Puedes instalarla ejecutando:
  ```bash
  pip install pandas
  ```

**Para ejecutar un archivo:**
Abre tu terminal, navega a la carpeta correspondiente y ejecuta el archivo deseado con Python. Por ejemplo:

```bash
python "Actividad V/8_desafio.py"
```
